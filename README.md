# Log Analyzer

Скрипт для анализа access.log файлов веб-сервера.

## Возможности
- Обработка одного лог-файла или директории с логами
- Подсчёт общего количества запросов
- Статистика по HTTP-методам
- Топ-3 IP адресов
- Топ-3 самых долгих запросов
- Сохранение статистики в JSON

## Запуск

```bash
python3 log_analyzer.py access.log
python3 log_analyzer.py /path/to/logs/
